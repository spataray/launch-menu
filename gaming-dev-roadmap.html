Why This Is Perfect

  • r/BaldursGate3 has 1.4M+ members constantly asking "what's the best build for X?"
  • He already knows D&D 5e mechanics from playing BG3 — domain expertise is the hard part
  • No good AI-powered build tool exists — current guides are static articles

How It Works

  • User describes what they want: class fantasy, playstyle, party role
  • Claude API analyzes D&D 5e rules + BG3-specific mechanics (fed via system prompt)

  • Patch notes fed as context so recommendations stay current

Tech Stack

What Overwolf Is

  • A platform specifically for in-game overlay apps — like browser extensions but for games
  • You build with web tech (HTML, CSS, JavaScript) — no game engine needed
  • They handle app store distribution, auto-updates, and ad monetization
  • Revenue share: 70-80% to you. Tracker.gg is built on this platform.

Overlay Ideas for His Games

  • Total War: Warhammer — unit counter-picker overlay during army selection
  • BG3-adjacent — D&D spell reference overlay for tabletop sessions streamed on Discord
  • General — AI gaming assistant overlay that answers questions about any game in real-time

The Revenue Math

  • Overwolf apps earn primarily through in-app ads (displayed between game sessions)
  • Average eCPM: $5-15 depending on region and game
  • 10,000 daily active users x $8 eCPM = ~$2,400/month
  • This is the most accessible path to serious passive income for a solo dev
