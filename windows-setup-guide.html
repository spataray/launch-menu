Step 2 — ~3 minutes

Install Git

Git tracks every change you make to your code. Install it with one command using Windows' built-in package manager.

PowerShell (Admin)
PS C:\> winget install --id Git.Git -e --source winget

Found Git [Git.Git] Version 2.47.1
Starting package install...
Successfully installed
⚠ Close and reopen Windows Terminal after installing Git so the git command is available in your PATH.
PowerShell
# Verify git is installed
PS C:\> git --version
git version 2.47.1.windows.1
wingetGitPATH
Step 3 — ~10 minutes (requires reboot)

Install WSL2 + Ubuntu

WSL2 (Windows Subsystem for Linux) runs a full Ubuntu Linux environment inside Windows. Claude Code CLI runs inside WSL. This is the biggest step — it requires a restart.

PowerShell (Admin)
PS C:\> wsl --install

Installing: Virtual Machine Platform
Installing: Windows Subsystem for Linux
Installing: Ubuntu
The requested operation is successful.
Please restart your machine to complete the installation.
🔄 Restart your PC now. After rebooting, Ubuntu will automatically open and ask you to create a username and password. This is your Linux user — pick something simple.

Step 5 — ~3 minutes

Install VS Code + WSL Extension

Visual Studio Code is the code editor you'll live in. The WSL extension lets it connect directly to your Ubuntu environment so you edit Linux files with a Windows GUI.

PowerShell
PS C:\> winget install --id Microsoft.VisualStudioCode

Successfully installed

5 Project Structure
First project — ~3 minutes

Create Your Projects Folder & Init First Repo

Set up a clean folder structure and initialize your first project. This is where all your code lives.

PowerShell
# Create your projects folder
PS C:\> mkdir ~\Documents\projects
PS C:\> cd ~\Documents\projects

# Create your first project
PS C:\Users\christopher\Documents\projects> mkdir my-first-bot
PS ...\projects> cd my-first-bot

# Initialize git + npm

Initialized empty Git repository
PS ...\my-first-bot> npm init -y
Wrote to package.json

PS ...\my-first-bot> claude

6 Bonus — API Keys for Gaming Projects

🔑 API Keys You'll Want for Gaming Projects

1

Discord Bot Token — Go to discord.com/developers/applications, create a New Application, go to Bot tab, click Reset Token, copy it.

2

Steam Web API Key — Go to steamcommunity.com/dev/apikey, enter a domain name (use localhost for dev), and get your key instantly.

3

Claude API Key (optional) — If you want to use the Claude API directly in your apps (not just Claude Code CLI), get a key from console.anthropic.com.

PowerShell — Setting up .env file
# Create a .env file in your project folder (NEVER commit this!)
PS ...\my-first-bot> New-Item .env

# Add your keys (open in VS Code or Notepad)
PS ...\my-first-bot> code .env
.env (example contents)
DISCORD_TOKEN=your-bot-token-here
STEAM_API_KEY=your-steam-key-here
ANTHROPIC_API_KEY=sk-ant-your-key-here
⚠ Never commit API keys to GitHub! Always add .env to your .gitignore file. Claude Code will do this automatically when it sets up your project, but double-check by running: echo ".env" >> .gitignore
