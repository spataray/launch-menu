Request Types
full_channel_build
Full Channel Build
All 6 pipeline stages. Customer receives the complete 6-file package via email.
niche_research
Niche Research Only
Stage 1 only. Returns niche research report with CPM data and video ideas.
reorder_scripts
Reorder Scripts
Returning customers use a 6-char reorder code to get new scripts that match their original channel voice. Codes stored in reorder-codes.json.

Key Files
File Role
poc/watchdog.sh Main pipeline orchestrator (~645 lines). All Claude agent prompts and parallelization logic.
index.html Landing page + animated 6-station pipeline status overlay (~1250 lines, vanilla JS).
pipeline Bash CLI wrapper: start / stop / restart / status / logs
poc/send-email.py HTML email assembly with file attachments via Gmail SMTP.
poc/poll-sheets.py Google Sheets CSV poller — creates JSON order files in submissions/pending/.
poc/google-apps-script.js Google Sheets webhook. doPost for submit, doGet for status polling.
poc/receiver.py Local HTTP server — alternative to Google Sheets for receiving orders.
